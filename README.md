# PlanuokFetch

A personal expense tracking iOS app built to practice and learn SwiftUI development.

## About

PlanuokFetch is a simple budget and expense tracker app developed as a learning project 
to explore SwiftUI, iOS development patterns, and REST API integration. The app allows 
users to manage their personal finances by tracking income and expenses across categories.

## Learning Goals

This project was built to practice:
- SwiftUI views and navigation
- MVVM architecture pattern
- Networking and API calls
- Unit testing in Xcode
